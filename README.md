# Pyqt5_highspeed
综述：利用python的pyqt库实现一个简洁的pyqt5界面

## 界面介绍
在login.ui 和 function.ui中是用pyqt5自带的图形化界面拖动控件生成的基础界面，其中控件的布局等需要自行设计和微调，具体方法参见网络
